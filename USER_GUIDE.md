## 📊 核心概念

### 💰 MLT (MemeLaunch Token)

```
遊戲唯一貨幣
起始資金: 10,000 MLT
用途:
  ├─ 創建幣種: 1,800-10,000 MLT
  ├─ 購買代幣: 任意數量
  └─ 賺取利潤: 賣出代幣換回 MLT
```

### 📈 Bonding Curve (聯合曲線)

```
定價機制: 指數增長
公式: Price = Initial × e^(4 × progress)

價格倍數:
0%  → 1.00×   (起點)
25% → 2.72×   (早期)
50% → 7.39×   (中期)
75% → 20.09×  (後期)
100% → 54.60× (畢業)

重點:
✅ 越早買入,成本越低
✅ 進度越高,價格越貴
✅ 賣出會降低進度
```

### 🤖 AI 交易員

```
5 種類型:
├─ SNIPER (狙擊手)
│  └─ 創建後立即買入 5-15%
├─ WHALE (鯨魚)
│  └─ 大額交易 10-30%
├─ RETAIL (散戶)
│  └─ 跟風交易 0.1-5%
├─ BOT (機器人)
│  └─ 高頻小額 0.1-1%
└─ MARKET_MAKER (做市商)
   └─ 雙向交易 1-10%

活動時機:
⚡ 創建後 0-5 分鐘: 狙擊手
🐋 5-30 分鐘: 鯨魚和散戶
🤖 持續: 機器人和做市商
```

```
每個幣創建時分配命運:

🛡️ SURVIVAL (5% 機率)

   1. 買入代幣
   2. 等待價格上漲

   - 每日登入獎勵

### Q3: 創幣最少需要多少 MLT?
```
A: 最少 1,800 MLT:
   ├─ 初始投資: 1,800 MLT
   └─ 預購代幣: ~100 MLT
   
   推薦 5,000+ MLT 以獲得:
   ├─ 更高起始進度
   ├─ 更多預購代幣
   └─ 更好的 AI 關注

### Q4: 為什麼我的幣沒有交易?
```
A: 可能原因:
   ├─ 命運是 EARLY_DEATH (會快速死亡)
   ├─ AI 交易員還未啟動 (等待 1-5 分鐘)
   ├─ 價格太高 (進度 > 80%)
   └─ Hype 分數太低
   
   解決方法:
   ├─ 等待 AI 交易員活躍
   ├─ 降低賣出價格
   └─ 提高 Hype (通過社交分享)
```

### Q5: 如何判斷是否是 Rug Pull?
```
A: 警告信號:
   ⚠️ 創建後立即大量賣出
   ⚠️ RUG_PULL_WARNING 事件
   ⚠️ 創建者 ID 是已知詐騙者
   ⚠️ 描述和社交鏈接是假的
   
   預防措施:
   ✅ 避免命運是 RUG_PULL 的幣
   ✅ 檢查創建者歷史
   ✅ 不在創建後立即買入大量
   ✅ 設置止損 (虧損 20%)
```

### Q6: 最高能賺多少倍?
```
A: 理論最大: 54.6× (100% 進度)
   實際情況:
   ├─ 狙擊: 2-5×
   ├─ 跟隨鯨魚: 5-15×
   ├─ 長期持有: 20-50×
   └─ 畢業 (稀有): 50-100×+
```

### Q7: 交易手續費是多少?
```
A: 統一 1% 手續費:
   ├─ 買入: 1% 的 MLT
   └─ 賣出: 1% 的 MLT
   
   例子:
   買入 1,000 MLT 的代幣
   └─ 實際扣除: 1,010 MLT
   
   賣出獲得 2,000 MLT
   └─ 實際收到: 1,980 MLT
```
