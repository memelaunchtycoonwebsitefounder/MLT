# 💰 MLT經濟模型設計

## 基礎設定

### 用戶初始餘額
- **10,000 MLT**

### 目標
- 允許用戶最多創建 **5個Meme幣**（最低配置，無保護功能）
- 保持經濟平衡

---

## 💵 定價結構

### 創幣成本
```
基礎創建費用：1,500 MLT
- 這樣用戶可以創建：10,000 ÷ 1,500 = 6.67個幣
- 調整為 1,800 MLT，可創建：10,000 ÷ 1,800 = 5.56個幣
- ✅ 最終定價：1,800 MLT/幣
```

### 幣種保護功能
```
單項保護：
- Revoke Freeze（凍結權限）：300 MLT
- Revoke Mint（鑄造權限）：500 MLT
- Revoke Update（更新權限）：400 MLT

全套保護包：
- Full Revoke Bundle：1,000 MLT（節省 200 MLT）
- 總價值：1,200 MLT → 優惠價：1,000 MLT
```

### 完整成本示例
```
1. 基礎幣種（無保護）：1,800 MLT
2. 單項保護幣種：1,800 + 300-500 = 2,100-2,300 MLT
3. 全保護幣種：1,800 + 1,000 = 2,800 MLT

用戶策略：
- 5個基礎幣：5 × 1,800 = 9,000 MLT（剩餘 1,000 MLT）
- 3個全保護幣：3 × 2,800 = 8,400 MLT（剩餘 1,600 MLT）
- 混合：2個全保護 + 2個基礎 = 5,600 + 3,600 = 9,200 MLT
```

## 📊 交易手續費
```
買入/賣出手續費：0.3%
- 收取的手續費 → 銷毀或平台金庫
- 例：購買 1,000 MLT 的幣種，手續費 3 MLT
```

## 🎁 MLT獲取方式
```
1. 註冊獎勵：10,000 MLT（初始）
2. 交易獎勵：每筆交易獲得 1-5 MLT
3. 持有獎勵：持有某幣種獲得MLT（可選）
4. 邀請獎勵：邀請好友獲得 1,000 MLT（可選）
```

## 📈 經濟平衡分析

### 用戶行為預測
1. **保守型**：創建1-2個高質量全保護幣（2,800 MLT/個）
2. **激進型**：創建5個基礎幣，快速測試市場（1,800 MLT/個）
3. **平衡型**：創建3個幣，部分添加保護（2,000-2,500 MLT/個）

### 平台收益
- 每個活躍用戶預計創建 2-3 個幣
- 平均消耗：5,000-7,000 MLT
- 剩餘MLT用於交易和其他活動
